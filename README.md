# 🗄️ MySQL Database Project

This project contains MySQL programs and queries designed to demonstrate database concepts such as table creation, data insertion, joins, constraints, and queries.

---

## 🚀 Features

- Database design using MySQL  
- Well-structured tables  
- CRUD operations (Create, Read, Update, Delete)  
- Use of joins, constraints, and indexes  
- Beginner-friendly SQL queries

- SQL

   USE my_project;
